Принцип роботи додатку Simplex Method Solver:
1. Ввести матрицю коефіцієнтів системи рівнянь у поля. Далі натиснути Solve. Результати обчислень будуть відобраються в останньому стовпці матриці. Елементи, напроти яких стоятимуть Х-кси - будуть розв'язками задачі. Якщо шукався мінімум, то коефіцієнт В першого рядка з протилежним знаком.
1.1 При формуванні вхідної матриці потрібно враховувати такі правила:
	- Якщо шукається максимум, то коефіцієнти беруться із знаком мінус (очевидно з рівняння)
	- Якщо шукається мінімум, то знаки коефіцієнтів цільової функції змінюються на протилежні. Також на протилежні міняються знаки кофецієнтів тих обмежувальних нерівностей, де знак >=.

2. При використанні файлового способу введення вхідних даних потрібно:
	- в перший рядок файлу вписати кількість стовпців матриці (небазисних коефіцієнтів + коефіцієнт В)	
	- в другий рядок файлу вписати кількість рядків матриці (базисних коефіцієнтів)
	- далі вписати всі коефіцієнти через один пробіл (кожен рядок матриці вводиться з нового рядка файлу)
